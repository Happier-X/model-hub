# v0.0.2 (2026-07-23)

## 修复

- 修复分组页「配置到 Pi」失败：Tauri 2 命令参数键改为 camelCase（`groupId`）。
- 同步修复模型榜单强制刷新参数键（`forceRefresh`），避免同类 IPC 传参错误。

## 变更

- 新安装默认代理端口由 `8080` 改为 `8888`。
- 已存在 `shell.json` 中的 `gateway_port` 配置不会被覆盖。
- README 与客户端文档中的默认 Base URL 示例同步为 `http://127.0.0.1:8888`。

## 安装与更新

- 安装包：GitHub Release 中的 NSIS `.exe`
- 校验：同 Release 的 `SHA256SUMS.txt`
- 应用内更新清单：`https://github.com/Happier-X/model-hub/releases/latest/download/latest.json`
