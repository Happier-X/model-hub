# v0.0.4 (2026-07-24)

## 新增

- 首页新增最近一次成功请求信息，展示最终成功的模型、分组、供应商和时间。
- 新增桌面悬浮状态条：可在设置页开启，持续展示最近成功的上游模型与代理状态；支持拖动并记住位置。

## 变更

- 「概览」更名为「首页」，导航语义更清晰。
- 将端口、数据目录、自动更新与桌面悬浮条等配置集中到设置页，首页更专注于运行状态和使用概览。

## 安装与更新

- 安装包：GitHub Release 中的 NSIS `.exe`
- 校验：同 Release 的 `SHA256SUMS.txt`
- 应用内更新清单：`https://github.com/Happier-X/model-hub/releases/latest/download/latest.json`
