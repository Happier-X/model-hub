# v0.0.3 (2026-07-24)

## 修复

- 取消熔断与 `auto_failover`：上游错误即按配置顺序故障转移，行为更可预期。
- 修复应用内「检查更新」可能因 Vue 代理 Tauri `Update` 对象触发私有成员错误的问题。

## 变更

- 渐进接入 `happier-ui`，统一可映射控件的视觉与交互。
- 业务对话框表单改用 TanStack Form，校验与提交状态更清晰。

## 安装与更新

- 安装包：GitHub Release 中的 NSIS `.exe`
- 校验：同 Release 的 `SHA256SUMS.txt`
- 应用内更新清单：`https://github.com/Happier-X/model-hub/releases/latest/download/latest.json`
