# PRD：移除全部鉴权（本地开放 API）

## 背景

Model Hub 现为**仅本机**桌面应用。用户明确不需要管理账号、JWT、客户端 API Key 等任何鉴权。

## 目标

- 管理 API `/api/v1/*` **无需** Bearer JWT
- 客户端 `/v1/*` **无需** `sk-modelhub-...`
- 前端删除静默登录、管理 Token 兜底、API 密钥页与鉴权闸门
- 文档改为「本机开放、默认 127.0.0.1」

## 范围

### 必须

- gateway-rust：取消管理/客户端鉴权中间件强制校验（可删 login 强制路径依赖）
- 前端：删除 `ensureAdminSession` 流程、设置页鉴权 UI、`GatewayGate` 鉴权分支、侧栏「API 密钥」
- 仪表盘客户端探测：无 Key 即可测 `/v1/models`
- 更新 README / 客户端文档 / 上手文档

### 不做

- 不引入新的账号体系
- 不把默认监听改为 `0.0.0.0`（仍默认本机）
- 可不立即发版（用户可再要求）

## 安全说明（写入文档）

本机 `127.0.0.1` 绑定下，同机进程可调用管理与客户端 API。若用户改为非本机地址，等于局域网开放，需自行承担风险。

## 验收

- [x] 网关 running 后无需登录即可使用渠道/分组/日志
- [x] `/v1/models` 无 Authorization 返回 200（有分组时）
- [x] UI 无管理 Token / API 密钥入口
- [x] cargo test / pnpm lint 通过
