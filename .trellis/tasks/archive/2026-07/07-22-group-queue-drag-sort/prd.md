# 分组队列拖拽排序

## Goal

在分组编辑页的故障转移队列中支持**拖拽排序**，降低长队列调整成本；保留「上移/下移」作为备用操作。

## Requirements

- R1：队列每条可拖动调整顺序。
- R2：拖动后 `form.items` 顺序立即更新，**仍需点保存**才持久化。
- R3：保留上移/下移按钮，行为与拖动一致。
- R4：拖动过程中不误触发表单提交；输入框与按钮可正常点击。
- R5：有简短中文提示（如拖动手柄说明）。
- R6：不新增重型依赖优先（原生 HTML5 DnD 或极轻封装均可）。
- R7：`pnpm typecheck` / `lint` / `build` 通过。

## Acceptance Criteria

- [x] 可拖动调整队列顺序。
- [x] 上移/下移仍可用。
- [x] 保存后顺序与表单一致。
- [x] 前端门禁通过。

## Out of Scope

- 分组列表本身拖拽。
- 跨分组拖拽。
- 自动保存。
