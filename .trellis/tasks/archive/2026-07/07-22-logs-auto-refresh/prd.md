# 日志页自动刷新

## Goal

日志页打开期间默认每 3 秒自动刷新，离开页面立即停止；保留手动刷新。

## Requirements

- R1：进入页面立即加载，并启动 3 秒定时刷新。
- R2：组件卸载时清理定时器。
- R3：已有请求未完成时跳过本轮，禁止重叠请求。
- R4：保留当前页与筛选条件，不强制跳回第一页。
- R5：显示自动刷新状态，可手动暂停/恢复。

## Acceptance Criteria

- [x] 每 3 秒刷新
- [x] 离开页面停止
- [x] 无并发重叠请求
- [x] 可暂停/恢复，手动刷新保留
- [x] typecheck/lint 通过
