# 分组队列按模型能力排序

## Goal

在分组编辑区提供「按模型能力排序」按钮，基于上游模型名启发式评分重排当前表单队列；保存前可继续拖拽/上下移动。

## Requirements

- R1：仅在前端本地根据模型名评分排序，不请求外部服务。
- R2：按钮只修改当前表单 `items` 顺序，不自动保存数据库。
- R3：同分数保持原相对顺序，保证排序稳定。
- R4：识别主流系列：Claude、GPT/OpenAI、Gemini、DeepSeek、Qwen、Llama、Mistral；未识别模型排在已识别模型之后。
- R5：每个条目显示简短能力标签/分数，避免黑盒。
- R6：单元测试覆盖主流模型排序与稳定性。

## Acceptance Criteria

- [x] 点击按钮后队列按能力分数从高到低重排
- [x] 未识别模型稳定排在识别模型之后
- [x] 同分稳定排序
- [x] 保存前可继续手动调整
- [x] typecheck/lint/test 通过
